\subsection{상세한 내용}
item의 정책을 수행하는 스케쥴러 쓰레드 클래스.

item의 정보 수집 주기 및 장애 정보 수집 주기를 검사하여, 수집 주기가 된 item에 대해서 정보 수집 쓰레드에게 정보 수집을 요청하고, 해당 아이템에 다음 정책을 설장하는 클래스.
