\subsection{상세한 내용}
수집된 메시지를 SMS Manager와 규약된 메시지 포맷으로 변환하는 클래스.

메시지 유형에 따라 다음 3가지 포맷으로 변환시킨다.\begin{enumerate}
\item 
PASSIVE :\par
 \$\{HOSTNAME\} YYYY-MM-DD HH:MI:SS\par
 \$\{ITEM\_\-CODE\} \$\{ITEM\_\-DESCRIPTION\}\par
 PASSIVE \$\{COMMAND\}\par
 \$\{DATA HEADER\}\par
 \$\{DATA\}\par
 COMPLETED\par
\item 
ACTIVE :\par
 \$\{HOSTNAME\} YYYY-MM-DD HH:MI:SS\par
 \$\{ITEM\_\-CODE\} \$\{ITEM\_\-DESCRIPTION\}\par
 ACTIVE\par
 \$\{DATA HEADER\}\par
 \$\{DATA\}\par
 COMPLETED\par
\item 
EVENT :\par
 \$\{HOSTNAME\} YYYY-MM-DD HH:MI:SS\par
 \$\{ITEM\_\-CODE\} \$\{ITEM\_\-DESCRIPTION\}\par
 EVENT \$\{CONDITION\} \$\{ELEMENT\} \$\{SEVERITY\} \$\{START/END\}\par
 \$\{DATA HEADER\}\par
 \$\{DATA\}\par
 COMPLETED\par
 \end{enumerate}
