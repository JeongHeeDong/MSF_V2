상세한 내용

수집된 메시지를 SMS Manager와 규약된 메시지 포맷으로 변환하는 클래스.

메시지 유형에 따라 다음 3가지 포맷으로 변환시킨다.

  1. PASSIVE :
    ${HOSTNAME} YYYY-MM-DD HH:MI:SS
    ${ITEM_CODE} ${ITEM_DESCRIPTION}
    PASSIVE ${COMMAND}
    ${DATA HEADER}
    ${DATA}
    COMPLETED
  2. ACTIVE :
    ${HOSTNAME} YYYY-MM-DD HH:MI:SS
    ${ITEM_CODE} ${ITEM_DESCRIPTION}
    ACTIVE
    ${DATA HEADER}
    ${DATA}
    COMPLETED
  3. EVENT :
    ${HOSTNAME} YYYY-MM-DD HH:MI:SS
    ${ITEM_CODE} ${ITEM_DESCRIPTION}
    EVENT ${CONDITION} ${ELEMENT} ${SEVERITY} ${START/END}
    ${DATA HEADER}
    ${DATA}
    COMPLETED
