상세한 내용

SMS Manager로부터 수신된 명령 코드를 구문 분석하고 처리하는 클래스.

SMS Manager로부터 수신된 명령 코드가 정해진 syntax에 맞는지 검사하고, 정해진 규약에 따른 명령 코드를 분석하여, 요청 되어진 명령어를 처리하는 클래스.
