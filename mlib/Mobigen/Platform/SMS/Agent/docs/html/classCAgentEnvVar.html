상세한 내용

SMS Agent가 기동하면서 필요한 모든 환경 설정 정보 및 item 정보를 관리하는 클래스.

system kernel 정보를 조회하기 위한 인터페이스(CKernelCore)
item code 정보를 관리하는 인터페이스(CAgentConfigVar)
SMS Agent 내부적으로 데이터를 주고받기 위한 각종 메시지 큐
SMS Agent session 상태 정보
Database session pool 정보 등을 관리한다.

참조하시요:
CKernelCore, CAgentConfig, CDBPool.
