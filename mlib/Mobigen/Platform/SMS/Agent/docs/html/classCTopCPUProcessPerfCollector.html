상세한 내용

CPU 점유율 순서로 Top N개의 프로세스 정보를 수집하는 클래스.

scagent.xml 파일에 다음과 같이 설정되어 있다면,
<instances>
<instance name="cpu|5">cpu|5</instance>
</instances>
CPU 점유율 상위 5개의 프로세스 정보를 수집하여 처리한다.
