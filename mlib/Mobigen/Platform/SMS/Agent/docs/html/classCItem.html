상세한 내용

수집 대상 item 정보 객체.

item code, 수집 주기(schedule), 장애 수집 주기(eschedule), 수집여부(enable), 상세수집여부(method) 수집 대상 instance, 장애 임계치 판단 condition 정보 등을 관리한다.
