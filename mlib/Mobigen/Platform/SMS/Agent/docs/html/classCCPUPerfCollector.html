CPU 성능 임계치 장애를 발생시키는 메쏘드. 임계치를 초과한 경우에는 장애를 발생시키고, 이전에 장애가 발생한 적이 있고, 임계치를 초과하지 않은 경우에는 장애를 해지시킨다.

매개변수들:
CItemCondition   pointer.
character   CPU name.
.   임계치를 넘은 경우는 true, 초과하지 않은 경우 false.
CPU   성능 정보를 저장하고 있는 구조체.

void CCPUPerfCollector::collect ( ) [virtual]
 

CPU 성능 정보를 수집하여 SMS Manager로 전송하기 위한 메시지 포맷으로 변환하여 메시지 큐로 전송하는 메쏘드.

CCollector으로부터 재구현된.
