상세한 내용

메모리 점유율 순서로 Top N개의 프로세스 정보를 수집하는 클래스.

scagent.xml 파일에 다음과 같이 설정되어 있다면,
<instances>
<instance name="memory|5">memory|5</instance>
</instances>
메모리 점유율 상위 5개의 프로세스 정보를 수집하여 처리한다.
