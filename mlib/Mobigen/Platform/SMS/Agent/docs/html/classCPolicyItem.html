상세한 내용

스케쥴러에 의해 수집 쓰레드에 수집 요청을 할때 필요한 정책 정보를 관리하는 클래스.

수집 요청 item, 수집 유형, 수집 시간, 수집 명령어 등의 정보를 담고 있는 정보 클래스이다.
