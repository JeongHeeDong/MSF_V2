상세한 내용

Item의 Instance 정보를 관리하는 클래스.

Instance name, Instance value 정보를 관리한다. 현재 Instance name, Instance value은 같은 정보로 설정 관리되어야 한다. scagent.xml 파일에서 당음과 같은 형식으로 저장된 정보를 기록한다. <instances> <instance name="test">test</instance> </instances>
